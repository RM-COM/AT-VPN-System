Дорожная карта ветки

Выполнено

  • Ветка помечена как read-only.
  • Роль upstream отделена от stable и implementation-контуров.

В работе

  • Только периодическое сравнение при необходимости.

Следующие шаги

Переносить идеи только через review и тесты в нашей ветке.

Проверки перед завершением этапа

  • Перед переносом идеи: diff, code review, staging, regression.
  • Без тестов upstream-изменения не попадают в main.

Стоп-условия

  • Не пушить в upstream/*.
  • Не использовать как документацию нашего форка.
  • Не заменять наш README upstream-текстом.
Правило проекта. После значимого исправления, debug-этапа, документационного блока или новой функции изменения фиксируются в документации, коммитятся и отправляются в GitHub. Stable-ветка принимает только проверенные изменения.
